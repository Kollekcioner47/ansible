создадим конфиг файл
vim ansible.cfg
в нём мы напишем:
[default] - это указанение на конфиг по умолчанию
host_key_checking = false - чтобы не спрашивал про отпечатки пальцев при первом подключении
inventory = ./hosts.txt - чтобы не надо было указывать файл invenory
сохраняем и выходим
проверяем работу
ansible all -m ping
должен отработать пинг-понг
